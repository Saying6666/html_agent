# fdu_022

## Round 1
Create a premium long-scroll single-file `src/index.html` for a **corporate website** called **Drift Archive**.
Concept: a photography portfolio studio website for developers and operators who also care deeply about craft, narrative sequencing, and premium creative credibility.
Audience: developers and operators with strong design taste, as well as brand teams seeking a thoughtful photography studio with process clarity.
Style direction: **Organic Warm** with references to human-centered creative brands, warm editorial product pages, and premium studio storytelling.
Component language: soft rounded modules, editorial case-study panels, warm testimonial blocks, image-led project sections, and refined CTA moments.
Color direction: oat, sand, clay, espresso, muted sage, and warm terracotta accents with grounded contrast.
Typography direction: friendly but premium display headings paired with warm contemporary body text.
Motion direction: gentle, human, and polished with soft reveal motion and tasteful state changes.
Build a complete page with these sections: navbar, hero, studio positioning, selected projects, service pillars, process, client outcomes, testimonials, FAQ, CTA, footer.
Required interactions: hover states, active states, carousel, counters, FAQ accordion.
The page should feel like a premium creative studio site rather than a plain corporate brochure or generic portfolio grid.

## Round 2
Deepen the layout and content density.
The hero should blend photographic atmosphere with a clear studio positioning statement and immediate trust-building credibility.
Add believable copy around campaign work, portrait systems, image sequencing, production process, client collaboration, licensing clarity, and art-direction confidence.
Balance trust-building with conversion through clear service framing, process explanation, and project storytelling with more warmth than a normal agency site.
Let the page alternate between image-led sections and more strategic credibility modules so it feels both beautiful and commercially serious.
Keep enough vertical depth for a long full-page screenshot, with multiple strong section transitions and no thin filler sections.

## Round 3
Polish all interaction states and compliance details.
Hover and active states should read clearly across image cards, CTA modules, controls, and contact sections.
Any carousel moments should feel elegant rather than gimmicky, counters should animate with clean timing, and the FAQ accordion should feel polished and easy to scan.
Ensure the layout is responsive for desktop and tablet widths while preserving the warm editorial studio feel.
Keep all CSS and JS inline inside `index.html`.
Do not use React, Vue, Svelte, jQuery, or any build step.
Do not reference local images, local fonts, local CSS, or local JS.
Use remote imagery only if needed and inline SVG icons.
Return only the final `src/index.html` content with no explanation.

## Round 4
Final refinement pass.
The result should feel art-directed, credible, and commercially polished while still being emotionally rich.
Avoid sterile agency copy, shallow image-led sections, and no-context case studies.
Ensure enough depth for a long full-page screenshot and visible interactive states throughout.
Return one complete self-contained `index.html`.
