# fdu_002

## Round 1
Create a premium long-scroll single-file `src/index.html` for a **corporate website** called **Maison Verre**.
Concept: a boutique hotel booking page designed for creative professionals who value atmosphere, curation, architecture, and elevated travel experiences.
Audience: creative professionals, art directors, founders, and design-conscious travelers.
Style direction: **Glassmorphism** with references to high-end hospitality sites, soft cinematic layering, and editorial luxury travel branding.
Component language: refined translucent cards, elevated booking widgets, soft segmented controls, premium content panels.
Color direction: smoky charcoal, soft ivory, misted champagne, muted rose-gold highlights, atmospheric image-led contrast.
Typography direction: elegant editorial serif for headings paired with sophisticated sans-serif body text.
Motion direction: smooth, luxurious, ambient, slightly cinematic.
Build a complete page with these sections: navbar, hero booking moment, room collection, amenities, image gallery carousel, signature experiences, metrics, testimonials, FAQ accordion, booking CTA, footer.
Required interactions: hover states, active states, carousel, counters, FAQ accordion.
The page should feel like a real premium hotel brand, not a generic travel template.

## Round 2
Deepen the page with richer hospitality storytelling.
The hero should include an aspirational headline, concise booking support copy, date/guest UI styling, and a striking visual composition.
Create room cards with layered glass panels, pricing hints, occupancy details, and feature chips.
Add sections for spa, dining, local culture, and curated stay packages.
Use a cinematic gallery carousel with strong image framing and elegant navigation states.
Add counters for suite count, guest rating, chef experiences, or neighborhood highlights.
Testimonials should sound like real hospitality reviews from creative travelers.
FAQ should cover check-in, cancellation, amenities, parking, transfers, and booking policy.
Keep the layout long, immersive, and screenshot-friendly.

## Round 3
Polish the interaction quality.
Make buttons, chips, room cards, nav links, and booking controls show strong hover and active feedback.
Ensure the carousel feels premium and intuitive.
Use counters that animate into final values once visible.
Make the FAQ accordion elegant, readable, and smooth.
Preserve the glassmorphism effect with careful contrast and readability.
Keep all CSS and JS inline in `index.html`.
No React, Vue, Svelte, jQuery, or build tools.
No local assets or extra files.
Use remote images only and inline SVG icons.
Return only the final HTML.

## Round 4
Final refinement pass.
The result should feel luxurious, art-directed, and emotionally rich while still being usable and conversion-oriented.
Avoid cheesy travel clichés, weak stock-template layouts, and low-contrast glass panels.
Ensure enough depth for a long full-page screenshot and visible interactive states throughout.
Return one complete self-contained `index.html`.
